# 第三章 词法分析  

## 词法分析的功能  

+ 词法分析：根据词法规则识别及组合单词，进行词法检查。  
+ 对数字常数完成数字字符串到（二进制）数值的转换。  
+ 删去空格字符和注解。  

+ 实现方案：
    1. 词法分析单独作为一遍：
        + 优点：结构清晰、各遍功能单一  
        + 缺点： 效率低  
    2. 词法分析作为单独的子程序
